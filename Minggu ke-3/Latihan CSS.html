Pemrograman Web

pemrograman sebuah website itu adalah proses pemrograman atau penyusunan program tertentu dalam bentuk tertentu yang nantinya bisa dibaca oleh web server. Web server akan memindai perintah dalam bahasa pemrograman tersebut lalu akan mengeksekusinya dan menjalankan perintah yang diminta sehingga nantinya akan muncul tampilan website seperti yang telah diprogram sebelumnya.

Hampir semua bahasa pemrograman bisa diterapkan dalam web programming. Namun dengan satu syarat yaitu bahasa pemrograman yang dimaksud dapat diakses di web server dan bisa dibuat HTML, CSS, XHTML, XML. Nah beberapa bahasa pemrograman yang cukup populer antara lain yaitu Java, PHP, Ruby on Rails, ASP.NET, ASP Classic, Perl, JSP, dan Python.
